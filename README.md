# Backend - E-Commerce App

This is the backend API for the E-Commerce application built with **Node.js**, **Express**, and **MongoDB**.

## 📌 Features
- User authentication (JWT)
- Product management
- Order management
- Admin dashboard support
- Image upload
- Role-based access control
